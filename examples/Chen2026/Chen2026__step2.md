## 2. 結構化重點摘要

**背景**
- 台海自 2020 年成為最不穩定的衝突熱點之一；主流以「事件驅動」解讀中國姿態（視為對台獨與美國平衡行為的回應）。

**研究問題**
- 2020 年後中國對台對抗性安全訊號，主要由「對外部刺激的反應」還是「北京自身地緣戰略再評估」驅動？

**核心論點**
- 中國對台姿態與「**百年未有之大變局**」的地緣戰略判斷（信「東升西降」＋國力自信）相關，非單純反應台/美。
- 將安全訊號區分為 **assertiveness（強硬）** 與 **coerciveness（脅迫）** 兩種，分屬不同邏輯。

**研究方法**
- LLM 縱貫內容分析：Google Gemini 2.0 Flash（few-shot + RAG）分類 **1,998 篇人民日報**（2016–2024）。
- 統計：fractional response model（quasi-binomial GLM + logit；Newey-West HAC SE）；輔以 Spearman、Granger、邊際效果。

**關鍵發現**
- 強硬 ← **民族主義／認同**敘事（sovereignty_claim、twidentity）。
- 脅迫 ← **國家實力**（capability）敘事，且「**東升西降**」（intconfig3）對強硬正相關、對脅迫**負**相關 → 大變局內部矛盾、非單一鐵板。
- 美國因素（rivalry、struggle2、us_tw）與脅迫正相關 → 台海實為「中—美—台」三角。

**貢獻**
- 理論：拆解中國修正主義的**內部異質性**，挑戰 Doshi／Friedberg 的「單一、一致破壞性」論。
- 方法：驗證 LLM + RAG 作為長時段官方論述內容分析的可複製工具。
